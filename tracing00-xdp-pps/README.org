# Install packages
```
$ apt install clang llvm libelf-dev libpcap-dev build-essential automake git
``
# Compile xdp tutorial
```
$ git clone https://github.com/danieldin95/xdp-tutorial.git
$ git checkout just-pps
$ ./configre
$ make 
```
